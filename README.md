# mintfall

A high-performance airdrop and token distribution program built on Solana. Designed for high-speed, large-scale token distribution with secure transaction validation.

### Prerequisites

**Node.js:** I recommend using [nvm](https://github.com/nvm-sh/nvm) to manage node instances.

### Installation

1.  Clone this repository:
    ```bash
    git clone <repository-url>
    cd mintfall
    git checkout main
    ```
2.  Install the project dependencies:
    ```bash
    npm install
    ```

### Usage

To initiate the token distribution:

```bash
node index.js
```
